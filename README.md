# Flappy Escape

**Лабораторна робота №2 — Основи скриптування**

---

## 1. Назва гри

**Flappy Escape**  
(Аркадна гра на виживання)

---

## 2. Основна дія (1 речення)

**«Керуй пташкою, вчасно підстрибуй і пролітай між трубами, намагаючись вижити якомога довше.»**

*(Коротко: таймінг + ухиляння для виживання.)*

---

## 3. Схема керування

**Пристрій: одна кнопка (tap / натискання)**

| Елемент | Опис |
|--------|------|
| **Space / ЛКМ / Tap** | Пташка робить ривок вгору |
| **Гравітація** | Постійно тягне пташку вниз |

Гравець контролює **вертикальний рух**, обираючи момент натискання.

---

## 4. Мета гри

- Пролітати між трубами, не врізаючись у них.
- Виживати якомога довше.
- Набирати очки за:
  - час виживання
  - успішне проходження труб
- Побити власний рекорд.

---

## 5. Правила

1. **Рух:** пташка постійно падає вниз під дією гравітації.
2. **Стрибок:** при натисканні пташка різко піднімається вгору.
3. **Перешкоди:** труби з’являються справа і рухаються вліво.
4. **Прохід:** між трубами є отвір, через який потрібно пролетіти.
5. **Зіткнення:** дотик до труби або землі → програш.
6. **Очки:** нараховуються за час виживання та/або проходження труб.
7. **Game Over:** після зіткнення гра зупиняється.
8. **Рестарт:** будь-яка кнопка запускає нову спробу.

---

## 6. Тиск — Обмеження — Наслідок

| Питання | Відповідь |
|--------|-----------|
| **Що створює напругу?** | Необхідність точного таймінгу стрибків і постійний рух труб. |
| **Що обмежується?** | **Контроль** — лише одна кнопка; **простір** — вузькі проходи між трубами. |
| **Наслідок помилки** | **Миттєвий програш (Game Over)** і необхідність почати заново. |

---

## 7. Core Loop (короткий опис)

1. **Старт сесії** — пташка з’являється, рахунок = 0.
2. **Геймплей** — гравець натискає, щоб контролювати висоту.
3. **Перешкоди** — труби з’являються і рухаються назустріч.
4. **Виживання** — гравець пролітає між трубами.
5. **Помилка** — зіткнення → Game Over.
6. **Рестарт** — нова спроба з метою побити рекорд.

Цей цикл формує **“one more try” loop** — бажання зіграти ще раз.

---

## 8. Стиль і атмосфера

- **Жанр:** аркадна endless-гра.
- **Візуальний стиль:** простий 2D (cartoon / мінімалістичний).
- **Гравець:** пташка.
- **Перешкоди:** труби різної висоти.
- **Фон:** нескінченний scrolling background.
- **Анімації:** нахил пташки залежно від швидкості.
- **Інтерфейс:**
  - поточний рахунок
  - рекорд
  - екран Game Over

---

## 9. Монетизаційна модель (ранні аркади: 25¢ за одну сесію)

- **Вартість гри:** 25 центів за одну спробу.
- **Сесія:** триває до першої помилки.
- **Мотивація гравця:**
  - побити власний рекорд
  - пролетіти більше труб
  - вижити довше
